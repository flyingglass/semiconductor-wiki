# 半导体周期

## 历史规律（1958-2018）

半导体行业68年历史中，存在铁律：**五年一个周期**，每次波浪式上行（不是下行或平衡）。

驱动力：每次高潮背后都有一个杀手级应用——台式机 → 笔记本 → 手机 → 互联网 → 移动互联网。一次一个。

## 谢志峰观点：周期消失（2018年起）

2018年以后，周期"不下来了，直接往上走，往下波动很少"。原因：**五个杀手级应用同时爆发**：

1. 物联网
2. 5G通信
3. 电动汽车
4. 太空探索
5. 人工智能 + 大数据

后果：长期供不应求。"今天嫌内存贵等两年再买？没有了，过两年20倍、30倍了。起码三五年下不来。"

## 供需稀缺程度排序（谢志峰 2026）

| 赛道 | 稀缺程度 | 原因 |
|------|---------|------|
| 光芯片 | 10年供不应求 | OCS从0到1刚完成，量产不确定 |
| 存储 | 5年供不应求 | 三星/海力士/美光寡头控产，保持饥饿 |
| CPU | 约1年平衡 | — |
| GPU | 已饱和 | 要多少给多少 |

## 反周期投资 vs 新常态

- 三星/台积电经典策略：低潮加倍投资 → 下一周期独占红利
- 新问题：如果周期消失了（不再有"低潮"），后来者更没有入场机会
- 后来者困境不变：知道该投但"口袋里没钱"

## 摩尔定律：从"到头"到"永生"

### 谢志峰1990年预测（错误）
- 当时结论：250纳米到头（隧道效应击穿，绝缘层变薄，开关短路）
- 理论正确但预测失败：工程师把电压从5V→3.3V→0.6V，场强不变，延续至今

### 当前判断（2026）
- **"工程师足够聪明，有生之年看不到摩尔定律尽头"**
- 10纳米后物理尺寸未再缩小，所谓7nm/5nm/3nm都是性能命名（"假的"）
- 英特尔技术比台积电先进两代，但"太老实不说话"
- 未来：新材料 + 新架构 + 立体堆叠 → 性能继续提升
