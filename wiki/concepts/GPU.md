# GPU（图形处理单元）

## 定义

Graphics Processing Unit，专为并行计算设计的处理器。与 CPU（少量强核心串行处理）不同，GPU 拥有数千个弱核心，适合大规模并行计算——如"一个猛男搬几十块砖 vs 几千个小学生每人搬两块，小学生更快"。

## 发展历程

- **1999 年**：英伟达发明 GPU 品类名称（GeForce 256），取代"显卡芯片"
- **2001 年**：GeForce 3 发布可编程着色器 + Cg 语言 → 真正的可编程 GPU，英伟达使命"加速计算"诞生
- **2006 年**：CUDA 发布 → GPU 可用于通用科学计算
- **2012 年**：AlexNet 用两块 GTX580（各 500$，共 1000$）训练出突破性结果
- **2022 年**：H100 发布，4 万$/颗

## CPU vs GPU

| | CPU | GPU |
|------|------|------|
| 核心 | 数个强核心 | 数千个弱核心 |
| 处理方式 | 串行 | 并行 |
| 类比 | 猛男一次搬几十块砖 | 几千小学生每人两块 |
| 适用场景 | 复杂逻辑、顺序任务 | 大规模矩阵乘法、图形渲染、AI 训练 |
