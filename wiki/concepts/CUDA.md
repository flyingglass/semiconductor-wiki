# CUDA

## 定义

Compute Unified Device Architecture，英伟达的并行计算平台和编程模型。允许开发者用 C/C++ 等语言直接在 GPU 上编写通用计算程序。免费但闭源，只能在英伟达 GPU 上运行。

## 发展历程

- **2006 年**：英伟达发布 CUDA，开始持续投入（累计 100 亿$+）
- **策略**：大学免费送显卡 + 培训 CUDA 编程 → 锁定学生用户（类似 Bloomberg 终端策略）
- **2012 年**：AlexNet 首次证明 GPU + CUDA 可用于深度学习
- **当前**：600 万+ 注册开发者，运行在 5 亿+ GPU 上

## 竞争壁垒

| 壁垒类型 | 具体表现 |
|----------|---------|
| 客户锁定 | 开发者转换成本极高，不愿换别的平台 |
| 规模效应 | 百亿研发摊在 5 亿+ GPU 上，单位成本极低 |
| 网络效应 | 用的人越多 → 应用越多 → 生态越有价值 → 越多人用 |

## 与中国半导体

华为昇腾推出类 CUDA 平台，通过少量代码适配即可兼容。在地缘政治保护下可能"猥琐发育"，但英伟达可通过版本更新打破兼容性。
