# 英伟达（NVIDIA）

## 概述

全球 GPU 和 AI 计算领导者，1993 年黄仁勋、Chris Malachowsky、Curtis Priem 共同创立。从 3D 显卡起家，经 GPU 时代、CUDA 生态建设，最终在 AI 时代成为全球市值最高公司之一。核心业务：计算和数据中心（~90%）+ 游戏显卡（~10%）。

公司名来自拉丁文 invidia（嫉妒）。黄仁勋提出"零亿美金市场"概念——做尚未存在的市场。

## 关键历史

| 时期 | 关键事件 |
|------|---------|
| 1993 | 成立，A 轮 200 万$（红杉 + Sutter Hill），估值 600 万$ |
| 1995 | NV1 发布后失败（内存降价 + DirectX 不兼容 + 技术路线错） |
| 1997 | RIVA128 绝地反击，4 月 100 万颗，Q4 盈利，"黄氏定律"诞生（6 月/代） |
| 1998 | 台积电成为主要供应商（黄仁勋手写信给张忠谋） |
| 1999 | 纳斯达克上市，创建 GPU 品类 |
| 2001 | GeForce 3 发布，可编程 GPU 诞生，加速计算使命确立 |
| 2006 | CUDA 发布，开始软硬一体生态建设 |
| 2008 | G84/G86 缺陷 + 金融危机，股价跌 80%（39→7$） |
| 2012 | AlexNet 大爆炸，黄仁勋豪赌 AI（四大下注） |
| 2016 | 第一台 DGX-1 送 OpenAI |
| 2019 | 70 亿$收购 Mellanox（击败英特尔） |
| 2020 | 数据中心收入首次超过游戏 |
| 2022 | H100 发布 + ChatGPT 引爆 AI 需求 |
| 2025 | Omniverse + Cosmos 物理 AI 平台 |

## 四道城墙

1. **CUDA 生态**：600 万+ 注册开发者，转换成本高 + 网络效应 + 规模效应
2. **数据中心**：三层客户锁定（程序员→采购→云客户），"人类已知最有粘性的商业模式之一"
3. **产业链**：台积电 CoWoS 产能锁定 + ODM 伙伴排他性
4. **黄仁勋的网络**："不叫护城河，叫网络。让所有人都能进来一起成功。"
